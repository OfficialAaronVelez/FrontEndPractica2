Coffee House

A static, client-side coffee shop demo built with HTML, Tailwind CSS, and vanilla JavaScript.

- Login: Landing page; session-based "login" redirects to home.
- Home: Featured coffees with add-to-cart and favorite buttons.
- Favorites: Stored in localStorage per user, user favorites appear first.
- Cart: Sidebar with line items, remove, and total, header badge shows count.
- Checkout: Place order shows a thank-you message and clears the cart.

No backend; data lives in sessionStorage and localStorage
